BASIC NAVIGATION


- h - move left
- j - move down
k-  - move up
- l - move right

-  - move forward one word
-  - move backward one word
-  - move to end of word

 - move to beginning of line
 - move to end of line
g - move to beginning of file
 - move to end of file

ODES
----
 - insert mode (before cursor)
 - insert mode (after cursor)
o - open new line below
O - open new line above
SC - return to normal mode

DITING
-------
- delete character
 d - delete line
 - yank (copy) line
  - paste after cursor
  - paste before cursor
  - undo
 rl+r - redo
